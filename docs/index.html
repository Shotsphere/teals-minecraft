TEALS Minecraft Project

Introduction

The TEALS Minecraft project is a set of lessons based on the Minecraft Forge modding package. We have customized this package to reduce the amount of overhead of modifying Minecraft, so that students can easily create new kinds of items, blocks, and entities in a custom Minecraft client.

Currently, our package supports the IntelliJ and Eclipse Java IDEs. Unfortunately, the package also requires full internet access for initial set up on each machine. We are currently investigating a way to work around that requirement, so that install could be done from a thumb drive on a computer without internet access. More news here as it happens.

For instructors who want to get started to evaluate the course, you can download the package from the download tab. Unzip the contents of this package into a development directory, and then run either install-eclipse.cmd or install-idea.cmd depending on your development environment. You will need full internet access to complete setup.

For questions, please contact Connor Hollasch <connor@hollasch.net> or Steve Hollasch <steve@hollasch.net>.

In addition, we have a project wiki everyone can use to share out information amongst instructors and students using this project.
